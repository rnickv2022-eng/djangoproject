Формат сдачи:

Сделать ДЗ в отдельной ветке и открыть Pull Request самому себе в main. На проверку отправить ссылку на PR.

1. Перейти в основную ветку (если ещё не в ней): `git checkout main`
2. Обновить её: `git pull`
3. Создать ветку для ДЗ: `git checkout -b <название_ветки>`, например, `git checkout -b homework-1`
4. Выполнить ДЗ
5. Добавить файлы в git: `git add -A`
6. Создать коммит: `git commit -m "комментарий к коммиту"`
7. Отправить изменения в github: `git push -u origin <название_ветки>`, например, `git push -u origin homework-1`
8. Открыть репозиторий и нажать на большую зелёную кнопку "Compare & pull request"
9. Написать краткое описание выполненного ДЗ
10. Нажать кнопку "Create Pull Request"
11. Скопировать ссылку и отправить на проверку
